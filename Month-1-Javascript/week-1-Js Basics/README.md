## Project  Completed 

###  1. FizzBuzz 

Implemented the classic FizBuzz challlenge using loops and condional logic 

### Key Learnings

- Understanding evaluation order within if/else-if statement 
- Identifying overlapping conditons and logic conflicts
- Testing edge cases  to validate logic
- Prioritizing specific conditions before general conditions

### Validation Scenarios

- Number divisble by 3 only
- Number divisible by 5 only
- Number divisible by both 3 and 5 
- Number divisible by neither

### Skill Practiced
